## Prerequisites

- [Docker](https://docs.docker.com/install/linux/docker-ce/ubuntu/)
- nginx
- a domain to use for the site
- optional: SSL certificate (outside scope of this README, use certbot for free certificates)

```sh
apt update
apt install -y nginx-light
```

## Usage

```sh
docker-compose up
```
